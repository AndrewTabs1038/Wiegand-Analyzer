# Overview
This is a plugin for the Salea logic-2 software for easy-to-use analysis of the Wiegand Protocol and allows for different size site, faculty, and card numbers, which the program also partitions. Data is displayed in a easy-to-read form, and also allows for multiple signals within the same time frame. Below, is an example of the plug-in being used.
